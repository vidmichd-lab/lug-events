# Дизайн-система

## Цветовая палитра

### Основные цвета
- **Primary (Blue)**: `#3b82f6` - основной цвет для кнопок и акцентов
- **Indigo**: `#6366f1` - дополнительный акцентный цвет
- **Purple**: `#a855f7` - для градиентов и декоративных элементов

### Градиенты
- Основной градиент: `from-blue-500 via-indigo-500 to-purple-500`
- Фоновый градиент: `from-slate-50 via-blue-50 to-indigo-50`

## Типографика

- **Заголовки**: Bold, градиентный текст для главных заголовков
- **Основной текст**: Regular, серый цвет (#374151)
- **Вторичный текст**: Medium, светло-серый (#6b7280)

## Компоненты

### Кнопки

#### Primary Button
```css
.btn-primary
```
- Градиентный фон (blue-600 to indigo-600)
- Белый текст
- Скругленные углы (rounded-xl)
- Тень и hover эффекты
- Анимация scale при наведении

#### Secondary Button
```css
.btn-secondary
```
- Белый фон
- Серый текст
- Граница
- Тень и hover эффекты

### Карточки

#### Event Card
- Белый фон с мягкой тенью
- Градиентная полоска сверху
- Hover эффект с поднятием и увеличением тени
- Адаптивная сетка

#### Info Card
- Градиентный фон (от светлого к более насыщенному)
- Иконка в цветном круге
- Скругленные углы (rounded-xl)

### Формы

#### Input Field
```css
.input-field
```
- Белый фон
- Скругленные углы (rounded-xl)
- Фокус: синее кольцо
- Плавные переходы

### Бейджи

- Скругленные (rounded-full)
- Цветные фоны с соответствующим текстом
- Иконки для визуального разделения

## Анимации

### Fade In
```css
animate-fade-in
```
Плавное появление элемента

### Slide Up
```css
animate-slide-up
```
Появление элемента снизу вверх

### Hover Effects
- Scale: увеличение на 5% при наведении
- Shadow: увеличение тени
- Translate: поднятие элемента на 4px

## Адаптивность

### Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### Grid System
- Mobile: 1 колонка
- Tablet: 2 колонки
- Desktop: 3 колонки

## Иконки

Используются SVG иконки из Heroicons:
- Outline стиль для большинства иконок
- Цвет соответствует контексту
- Размер: 4-6 (w-4 h-4 до w-6 h-6)

## Тени

### Soft Shadow
```css
shadow-soft
```
Мягкая тень для карточек и контейнеров

### Glow Shadow
```css
shadow-glow
```
Свечение для акцентных элементов

## Состояния загрузки

- Spinner: анимированный круг
- Skeleton: пульсирующие серые блоки
- Empty State: иконка + текст + CTA кнопка

## Пустые состояния

Все пустые состояния включают:
- Крупную иконку
- Заголовок
- Описание
- Кнопку действия (если применимо)
